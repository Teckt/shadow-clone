How it Works
  1. You provide repository and issue details
  2. Our Flask app calls the MCP GitHub tool
  3. The tool assigns the issue to copilot-swe-agent
  4. Copilot starts working on the issue
  5. You can track progress in real-time

Flow:
  1. User Request: Submit task via web interface
  2. Flask Processing: Validate and prepare request
  3. MCP Integration: Call appropriate GitHub MCP tool
  4. GitHub API: Execute GraphQL mutations to assign Copilot
  5. Copilot Activation: GitHub Copilot coding agent starts working
  6. Real-time Updates: Track progress and provide feedback
